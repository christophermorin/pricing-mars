## About The Project

[![Pricing Mars][product-screenshot]](https://pricing-mars.vercel.app/)

A slightly tounge in cheek project, content wise, that I built after running into multiple job applications listing Sass as a preferred styling technique. Having never used Sass before I decided to build this pricing page to gain some experience and have a bit fun while doing so.
